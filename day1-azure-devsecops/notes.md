# Day 1: Azure DevSecOps Fundamentals

### Key Learnings:
1. Azure DevOps Environment: Understood how to set up Organizations and Projects in Azure DevOps.
2. Agent Configuration: Successfully configured and registered a *Self-hosted Windows Agent* for running pipelines locally.
3. Pipeline as Code: Learned how to define build pipelines using and the importance of azure-pipelines.
4. Git Version Control: Practiced advanced Git operations including branching, remote management, and force pushing to specific sub-folders.
5. Security Integration: Explored the basics of integrating security tools (like SonarCloud) into the CI/CD workflow to ensure code quality.
